# 🚗 Калькулятор доходности автомобиля

Веб-приложение для расчёта инвестиционной доходности автомобиля, сданного в аренду.

## ✨ Возможности

- Расчёт месячного оборота, комиссии, операционных расходов
- Чистая выплата инвестору в месяц
- Срок окупаемости (в месяцах и годах)
- Годовая доходность (ROI)
- Визуализация структуры доходов и расходов
- Экспорт отчёта в PDF
- Полностью адаптивный дизайн (мобайл + десктоп)

## 🛠 Технологии

| Технология | Назначение |
|---|---|
| Next.js 14 (App Router) | Фреймворк |
| TypeScript | Типизация |
| TanStack React Query | Серверное/клиентское состояние |
| Tailwind CSS | Стилизация |
| jsPDF + jspdf-autotable | Генерация PDF |
| Docker + Docker Compose | Контейнеризация |

## 🚀 Запуск

### Вариант 1 — Локально (Node.js)

```bash

npm install

# 2. Запустить dev-сервер
npm run dev
```

Открыть: [http://localhost:3000](http://localhost:3000)

### Вариант 2 — Docker Compose (продакшн)

```bash
# Собрать и запустить

# В фоне
docker compose up --build -d

# Остановить
docker compose down
```

Открыть: [http://localhost:3000](http://localhost:3000)

---

### Вариант 3 — Docker Compose (разработка, с hot reload)

```bash
docker compose --profile dev up app-dev --build
```

## 🧮 Формулы расчёта

```
Месячный оборот       = цена_аренды × дней_в_месяц
Комиссия компании     = оборот × (комиссия% / 100)
Доход после комиссии  = оборот − комиссия
Чистая выплата        = доход_после_комиссии − операционные_расходы
Срок окупаемости      = стоимость_авто / чистая_выплата
Годовой доход         = чистая_выплата × 12
ROI                   = (годовой_доход / стоимость_авто) × 100
```

### Пример — Lamborghini Urus

| Параметр | Значение |
|---|---|
| Стоимость | $320,000 |
| Аренда/день | $1,200 |
| Загрузка | 18 дней/мес. |
| Комиссия | 40% |
| Расходы/мес. | $5,500 |
| **Чистая выплата** | **$7,460/мес.** |
| **Окупаемость** | **~42.9 мес. (3.6 года)** |
| **ROI** | **~28% годовых** |
